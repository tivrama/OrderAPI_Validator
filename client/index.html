API Validator

Welcome to the API Validator. Submit your post body to check for required fields and correct schemas. The exact payload will be saved for reference. You then get a reponse json which will point out any errors, missing or malformed data, and notes

Use this for the following products: return, alert, monitor, ship

Get a validation of a JSON payload:

POST: https://api-validator.herokuapp.com/api/validator?retailer=&product=

Example: https://api-validator.herokuapp.com/api/validator?retailer=peninsula&product=return
